# IoTHarborAnywhere

IoT Harbor service, one can take anywhere! - initially based on the Odroid H2+, Debian 10 and 7error's Harbor repo

## Initial Hardware

Odroid H2+ \
32Gb DDR4 memory \
256Gb M.2 NVME SSD for the OS \
2T Samsung 960 2.5inch SSD \
